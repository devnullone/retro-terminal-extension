# Last Clicker Retro Terminal Extension

A cross-browser (Chrome & Firefox) popup extension that provides a retro terminal interface for the Last Clicker EIP-7702 game. Access the full game experience in a nostalgic terminal aesthetic.

## Development Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Dev Mode (Hot Reloading):**
    *   Note: Hot reloading works for the UI in a standard browser tab, but for the actual extension popup, you usually need to rebuild or use specific vite extension plugins.
    *   Run `npm run dev` and open the localhost link to design the UI.

3.  **Build for Production:**
    ```bash
    npm run build
    ```
    This creates a `dist` folder containing the extension.

## Installation

### Chrome / Edge

1.  Open `chrome://extensions`.
2.  Enable **Developer mode** (top right).
3.  Click **Load unpacked**.
4.  Select the `dist` folder generated by `npm run build`.

### Firefox

1.  Open `about:debugging#/runtime/this-firefox`.
2.  Click **Load Temporary Add-on...**.
3.  Navigate to the `dist` folder and select the `manifest.json` file.

## Icons

The `manifest.json` references icons in `icons/`. For the build to work perfectly without errors in the extensions manager, place PNG files (16x16, 48x48, 128x128) into `public/icons/`.

## Firefox Compatibility

This extension is now fully compatible with Firefox's Content Security Policy (CSP) requirements:
- ✅ **Local fonts**: VT323 font bundled locally (no external font loading)
- ✅ **No external CDNs**: All CSS and JS bundled internally  
- ✅ **Processed Tailwind**: All utility classes pre-compiled
- ✅ **Self-contained**: No external resource dependencies

## Notes on Dimensions

The app is styled for 450px width per the design rules. Chrome limits popup height to approximately 600px. If content exceeds this, scrollbars will appear. Firefox may allow slightly larger dimensions but also has limits.

## Design Rules Compliance

All styling follows the retro terminal aesthetic specified in `.copilot/rules.md`:
- **Background**: Black (#000000)  
- **Text**: White (#FFFFFF)
- **Highlights**: Yellow (#FFFF00)
- **Font**: VT323 monospace (Google Fonts, bundled locally)
